## 🎓 LESSONS FROM COMPETITORS

### **Fresha: Freemium + Marketplace Model**

**What They Did Right**:
```
├── FREE base product → Massive adoption
├── 20% commission on marketplace bookings → Revenue
├── Payment processing → Additional revenue
└── Raised $317.5M → Scale globally
```

**Balik.Lagi Application**:
```
├── Year 1-2: Focus on PAID subscriptions (Indonesia market)
├── Year 3+: Launch marketplace "Cari Barbershop Terbaik"
├── Monetize: 10-15% commission new customers
└── Future: Payment integration (partner with Midtrans/Xendit)
```

### **Majoo: All-in-One Approach**

**What They Did Right**:
```
├── POS + Accounting + Payroll → Sticky product
├── 30,000+ SMEs Indonesia → Local market expertise
├── $79.6M revenue → Profitable scale
└── Tier pricing → Capture all segments
```

**Balik.Lagi Application**:
```
├── Specialized > General (barbershop ONLY)
├── Deep features > Wide features
├── Better fit > More features
└── Niche domination > Horizontal expansion
```

### **Squire: Premium Positioning**

**What They Did Right**:
```
├── $750M valuation → Investor confidence
├── Premium brand → Higher pricing power
├── Enterprise focus → Bigger contracts
└── Built for barbers → Authentic positioning
```

**Balik.Lagi Application**:
```
├── Start: Affordable (Indonesia purchasing power)
├── Later: Premium tier for chains
├── Always: "By capster, for capster" authenticity
└── Brand: Emotional (Balik.Lagi) > Technical (OASIS)
```
